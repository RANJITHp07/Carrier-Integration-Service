# Carrier Integration Service

Simple TypeScript + Express scaffold.

Quick start

1. Install dependencies:

```bash
npm install
```

2. Run development server:

```bash
npm run dev
```

3. Build and run production:

```bash
npm run build
npm start
```

Files added: `package.json`, `tsconfig.json`, `src/index.ts`, `.gitignore`.
